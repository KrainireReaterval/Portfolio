## Step 3: What Each File Does (Quick Reference)

**Files you'll edit often:**
- `src/app/page.tsx` - Your homepage content
- `src/app/layout.tsx` - Site-wide layout (navigation, footer)
- `src/components/*` - Reusable UI pieces
- `public/images/*` - Your images

**Configuration files (edit once):**
- `src/lib/seo/config.ts` - SEO settings
- `tailwind.config.ts` - Design system colors/fonts

**Files you rarely touch:**
- `package.json` - Dependencies list
- `tsconfig.json` - TypeScript settings
- `next.config.ts` - Next.js settings

## üé® Quick Win: Let's Make a Simple Change

Try this yourself:

1. Open `src/app/page.tsx`
2. Find the heading text
3. Change it to your name
4. Save the file
5. Watch it update in the browser automatically!

This is called **Hot Module Replacement** - changes appear instantly without refreshing.
